## Термины и технологии

| Термин | Объяснение |
|---|---|
| Hydration mismatch (расхождение гидратации) | Ситуация в React/Next.js, когда HTML, отрисованный на сервере, не совпадает с тем, что React рендерит на клиенте при первой загрузке — вызывает предупреждения в консоли и визуальные "прыжки" интерфейса. На этой неделе несколько таких багов нашли и исправили (тема, панели портфолио). |
| `useSyncExternalStore` | Хук React для безопасной (без гидратационных багов) подписки компонента на внешнее хранилище состояния — например, `localStorage`. Используется в переписанном `ThemeProvider.tsx`, чтобы тема сайта не расходилась между сервером и клиентом. |
| `suppressHydrationWarning` | Специальный атрибут React, который отключает предупреждение о несовпадении разметки для конкретного элемента — применён на `<html>`, потому что блокирующий скрипт темы намеренно меняет класс `<html>` ещё до того, как React успевает отрисовать страницу. |
| Специфичность CSS и `:where()` | `:where()` — CSS-селектор с нулевой "специфичностью" (весом при конфликте правил). Tailwind использует его для варианта `dark:`, из-за чего вручную объявленные классы иногда "перебивали" тёмную тему — пришлось выносить цвета колец `AuthorCta` в отдельный CSS без `:where()`. |
| Линейная интерполяция/экстраполяция | Способ вычислить промежуточное (или заходящее за пределы) значение между двумя известными точками по прямой линии — так `BuildEstimator.tsx` считает срок стройки между опорными точками (150→7 дней, 400→14 дней, 600→24.5 дня). |
| Frankfurter API | Бесплатный API курсов валют (данные ЕЦБ, без ключа доступа, разрешает запросы прямо из браузера) — источник живого курса в калькуляторе на `/contact`. |
| `flexGrow`-анимация vs `layout`-анимация Framer Motion | В Framer Motion режим `layout` умеет "подделывать" изменение ширины блока через CSS `transform: scale`, что визуально растягивает фото внутри. Настоящая анимация числового `flexGrow` вызывает честный reflow (пересчёт вёрстки браузером) — дороже по производительности, зато картинка не искажается. |
| Emoji-флаги и рендеринг на Windows | У emoji-флагов стран нет гарантии, что операционная система умеет их отрисовывать как флаг — на Windows без нужного шрифта вместо флага показывается буквенный код страны. Из-за этого в калькуляторе валют флаги заменили на текст, но в `ClientReviews.tsx` та же проблема пока не исправлена (см. "Проверка структуры"). |
| `group` / `group-hover` (Tailwind) | Приём Tailwind: класс `group` на родительском элементе позволяет дочернему элементу реагировать на hover/состояние родителя через `group-hover:`/`group-open:` — используется, чтобы стрелка внутри кнопки вращалась при наведении на саму кнопку, и чтобы иконка `+`/`-` в FAQ менялась при открытии вопроса. |
| Vercel Preview-деплой и ветка `dev` | Промежуточная ссылка, которую Vercel автоматически собирает для непродакшен-веток (в проекте — `dev`), чтобы можно было проверить изменения вживую до мёржа в `master` (который деплоится на реальный домен). |
